# Terrain sprite assets (Track A)

Place transparent PNGs here named `type_WxH.png`, e.g.:

- `grass_64x64.png` — seamless ground tile
- `dirt_64x64.png` — seamless path/ground tile
- `rock_64x64.png`
- `bush_48x48.png`
- `tree_128x160.png` (+ `tree_b_128x160.png` variant)

See `client/src/game/terrain/CONTRACTS.md` for the full asset + manifest contract.
